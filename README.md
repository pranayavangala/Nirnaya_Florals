# Nirnaya Florals — React scaffold

A minimal Vite + React starter created for you.

Run the project:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Files added:
- `package.json` — scripts and deps
- `index.html` — app entry
- `src/main.jsx`, `src/App.jsx`, `src/index.css`

Edit `src/App.jsx` to start developing.
